Acceso seguro a la información

Uno de los puntos más importantes en cualquier sistema empresarial es controlar quién puede acceder a qué información. Si todos los usuarios pudieran ver y cambiar cualquier cosa sería un problema grave para la empresa. Por eso Odoo tiene un sistema de usuarios, roles y permisos.

El acceso a Odoo siempre se hace con usuario y contraseña desde la pantalla de inicio de sesión. Cada usuario tiene asignado un rol que determina a qué partes del sistema puede acceder.

Gestión de usuarios

Para gestionar los usuarios en Odoo hay que ir a Ajustes y luego hacer clic en Administrar usuarios. Desde ahí podemos ver todos los usuarios que existen, crear nuevos o modificar los que ya hay.

Lista de usuarios en el panel de administración de Odoo

Crear un usuario nuevo

Para crear un usuario nuevo hacemos clic en Nuevo. Tenemos que rellenar el nombre y el correo electrónico que será su nombre de usuario para iniciar sesión. También le asignamos el tipo de acceso que va a tener.

Tipos de acceso y roles

Odoo tiene tres niveles principales de acceso que se asignan a cada usuario:

Tipo de usuario Descripción Qué puede hacer
Administrador Tiene acceso total al sistema. Puede hacer cualquier cosa: crear usuarios, instalar módulos, cambiar configuraciones.
Usuario interno Usuario normal de la empresa. Puede usar los módulos que el administrador le haya dado acceso.
Portal Acceso limitado para clientes o proveedores externos. Solo puede ver sus pedidos, facturas o tickets que le correspondan.

Permisos por módulo

Dentro de cada usuario se pueden configurar los permisos por módulo. Por ejemplo un usuario puede tener acceso al módulo de Ventas pero no al de Contabilidad. Esto se hace desde el formulario del usuario en la sección de permisos.
